# healthcare-executive-dashboard-powerbi

Power BI healthcare executive dashboard for hospital KPI monitoring, claims cost analysis, readmission risk analysis, ICU utilization, and healthcare performance monitoring.

---

## Dashboard Features

- Executive KPI overview
- Readmission risk analysis
- ICU utilization monitoring
- Cost analysis by diagnosis
- Hospital performance tracking
- Patient segmentation
- Claims analytics

---

## Technologies Used

- Power BI
- DAX
- SQL
- Python-generated healthcare data
- Healthcare analytics

---

## Dashboard Pages

### Executive Overview
- Total claims cost
- Average hospitalization cost
- Readmission rate
- ICU admission rate

### Hospital Performance
- Average length of stay
- Cost by hospital
- Readmission trends

### Risk Analytics
- High-risk patients
- Chronic disease analysis
- ICU risk monitoring

### Diagnosis Cost Analysis
- Cost by diagnosis
- Top expensive diagnoses
- Claims volume analysis
