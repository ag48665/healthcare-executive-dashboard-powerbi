## Project Overview

This project presents an interactive Power BI dashboard designed for healthcare executives and hospital management teams. The dashboard enables monitoring of healthcare KPIs, claims costs, readmission risks, ICU utilization, diagnosis trends, and hospital operational performance.

The project demonstrates healthcare analytics, business intelligence reporting, KPI monitoring, and dashboard development using Power BI, SQL, and Python-generated healthcare datasets.
